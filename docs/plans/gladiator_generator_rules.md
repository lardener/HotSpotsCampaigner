# GLADIATOR GENERATOR

The following system can be used to generate an opponent for a gladiator duel, based on the circuit being competed in. This system does not create opponents for class one—exoskeleton. First, roll 2D6 for Experience Rating. Subtract two from the roll for light and add two to the roll for heavy. Add 6 to the roll for class six—open. Once the Experience Rating is generated, you roll on the Gladiator Skill Levels Table to get specific Piloting and Gunnery Skills.

Our random gladiator needs a 'Mech unit. First, we roll to determine the gladiator's faction. This will determine the units available. After determining the gladiator's faction, use the related Random Assignment Table (see p. 182–185) matching the era and faction to determine the gladiator's 'Mech. While there have been open circuit MechWarriors using heavy 'Mechs, for the purposes of randomly generating a 'Mech, use the assault class RAT for the open circuit.

Special Pilot Abilities are a great way to distinguish MechWarriors, beyond just good or great. For a random gladiator, roll 1D6 and compare the result to the column for the experience level of the gladiator on the Gladiator Special Pilot Abilities Costs Table. Each number is a roll on the Gladiator Edge Ability Table An Elite gladiator that rolls a 3 on 1D6 would get two rolls on the Edge Ability Table. Reroll any results that are not appropriate for the unit (Jumping Jack when the 'Mech has no jump jets).
